# Telugu Movie Recommendation System using SVD

## Overview
This project builds a recommendation system for Telugu movies using 
Singular Value Decomposition (SVD).

The system analyzes user ratings and predicts movies that a user is 
likely to enjoy.

## Dataset
- Telugu movie dataset
- User ratings matrix

## Methodology
1. Data preprocessing
2. Creation of user–movie rating matrix
3. Applying SVD decomposition
4. Predicting missing ratings
5. Generating top movie recommendations

## ResultS
Suggest the movie for deired user

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
